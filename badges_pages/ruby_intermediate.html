What is it for?

For being able to write and run Ruby scripts that take advantage of more useful aspects of the language that increase efficiency and complexity of what can be achieved.

What skills does it represent?

This badge covers several programming concepts in the Ruby language:

  • defs and blocks
  • Using gem for ruby gem installation
  • defining classes and using instances of self-written classes

Key Skill

The bearer is able to write Ruby code to model and solve a scientific problem efficiently and effectively and take advantage of the vast body of freely available, pre-written Ruby code in gems.
